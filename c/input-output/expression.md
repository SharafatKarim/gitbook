## Introduction

Expressions in C are combinations of variables, constants, and operators that are evaluated to produce a value.

### Example: Arithmetic Expression

```c
#include <stdio.h>

int main() {
    int a = 5, b = 3;
    int sum = a + b;
    printf("Sum: %d\n", sum);
    return 0;
}
```
